Link to play the CivClicker game for yourself. 

The bot will automatically create a Chrome windows and go to the website to play it. 


To run the bot:

You will need to download the bot.rar file and extract the bot.exe file and run it.
Just make sure you are running the latest version of Chrome (Version 80.x)
